# 📊 Excel Database Viewer — AndromedaPVP

O **Excel Database Viewer** é uma ferramenta web interna desenvolvida para o projeto legado **AndromedaPVP**. O seu principal objetivo é facilitar a visualização e gestão de todas as informações dos jogadores que estavam armazenadas em bases de dados SQL, MySQL... os mesmos foram convertidos para Excel (.xlsx / .csv), centralizando-as numa interface web intuitiva e de rápida consulta.

---

## 🚀 Funcionalidades

* **Leitura Automatizada:** Importação e processamento de folhas de cálculo Excel.
* **Visualização Centralizada:** Exibição detalhada do perfil do jogador (estatísticas, histórico, economia, dados gerais, etc...).
* **Filtros e Pesquisa Rápida:** Procura instantânea por Nickname.

* Ainda por vir:
* **Interface Web Responsiva:** Painel administrativo limpo e fácil de navegar por qualquer membro da equipa de staff.
* **TOP Jogadores:** Na página inicial, principalmente de horas jogadas em geral. Mais horas => mais dados variados e completos.

---

## 🛠️ Tecnologias Utilizadas

> **Nota:** Este projeto faz parte do ecossistema *legado* do AndromedaPVP.

* **Frontend:** HTML5, CSS3 (Tailwind CSS / Bootstrap) e JavaScript.
* **Backend:** .
